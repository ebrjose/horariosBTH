# Reto Horarios

### Backend API

Para el backend se necesita tener instalado PostgreSQL

#### Instalacion
```sh
$ cd horarios-api
$ npm install
$ npm run start-dev
```
La api corre en el en el puerto **6801**  [http://localhost:6801/api-v1/](https://localhost:6801/api-v1/) 

### Frontend

El frontend corre con el Framework Vue.js

#### Instalacion
```sh
$ cd horarios-web
$ npm install
$ npm run dev
```
